# IAppService.TryImportModule method

Try to import a mpk file as a module to the app. If the module with the same name exists, user will get a dialog asking to replace it or to rename the importing module.

## Return Value

Flag indicating whether import was successful.

## Exceptions

| exception | condition |
| --- | --- |
| ArgumentNullException | *model* or *moduleMpkPath* is null or empty. |
| ArgumentException | *moduleMpkPath* does not exist. |
| NotSupportedException | *model* is not the one received from Studio Pro API. |
| InvalidOperationException | *model* has a pending transaction which is not allowed. |

## Remarks

We advise you to use temporary directory for *moduleMpkPath* and remove the file after import.
